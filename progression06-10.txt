Week 6

ฝึกเขียน Higher-order Function
ใช้ function เป็น argument เช่น greet(name, formatter)
สร้าง Nested Function และ Nested Arrow Function
เข้าใจ Scope และ Shadowing ของตัวแปร
สร้าง Closure เพื่อให้ function จำค่าได้ เช่น makeAdder idGenerator
คืน function จาก function เช่น outerFunction(a)(b)
ฝึกจัดการข้อมูลใน array ของ object
สร้าง function สำหรับเพิ่ม ลบ แก้ไข และดึงข้อมูลจาก array
ใช้ for loop เพื่อค้นหาและอัปเดตข้อมูลใน array ตาม id

Week 7

ฝึกใช้ document.documentElement document.firstChild และ document.firstElementChild
เข้าใจความต่างของ childNodes (รวม Text และ Comment Node) และ children (เฉพาะ Element Node)
ใช้ document.createElement("p") แล้วเพิ่มด้วย divElement.appendChild(pElement)
ใช้ textContent สำหรับข้อความล้วน และ innerHTML สำหรับเนื้อหาที่มี Tag HTML
ใช้ nodeType เพื่อระบุว่าเป็น Element Attribute หรือ Text Node
เปรียบเทียบ innerHTML innerText และ textContent

Week 8

เลือก Element เดี่ยวโดยใช้ getElementById หรือ querySelector เพื่อเลือก Element แรกที่ตรงเงื่อนไข
เลือก Element กลุ่ม ใช้ querySelectorAll (NodeList) และ getElementsBy (HTMLCollection)
จัดการ Collection ระหว่าง NodeList (ใช้ forEach ได้) กับ HTMLCollection (ต้องใช้ Array.from() ก่อน)
ลบ Element ใช้ parentElement.removeChild(childElement)
เพิ่ม Element ใช้ createElement และ parentElement.insertBefore(new, reference)
แทนที่ Element ใช้ createElement และ parentElement.replaceChild(new, old)
